# CareConnect — Mini Healthcare Support Web App

A concept-level healthcare support web app built for an NGO use-case.

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Static frontend deployment

## Features
- Patient support / volunteer registration form
- Smooth responsive UI
- Request type selection
- AI-style symptom/request summarization
- Priority detection based on keywords
- Suggested NGO volunteer actions
- Disclaimer that output is not medical advice

## AI / Automation Idea
The app includes a simple AI-style automation concept that reads the user's support request description and extracts important keywords such as fever, cough, breathing issue, pain, weakness, emergency, medicine, and clinic.

Based on detected words, it generates a request summary, priority level, and suggested NGO action steps.

## NGO Use Case
NGOs often receive many patient support requests through phone calls, forms, or WhatsApp. This prototype shows how a simple digital form with automation can help collect patient details, classify requests, summarize important information, and help volunteers respond faster.

## How to Run Locally
Open `index.html` directly in your browser.

No backend or database is required.
